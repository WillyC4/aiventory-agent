# AIventory Agent

AIventory Agent es un asistente conversacional para inventarios.  
Permite hacer preguntas en lenguaje natural y obtener respuestas claras sobre el estado del inventario.

## Características
- Responde en español de forma amigable.
- Convierte preguntas en consultas SQL seguras.
- Explica los resultados de manera sencilla.
- Detecta si la interacción es charla general o consulta de base de datos.

## Uso
Ejecuta el agente y escribe tus preguntas:

```bash
python agent.py
